# Digital Marketing - NexaStudio

My first frontend project — a digital agency landing page built with HTML, CSS, and JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js (for backend/contact form integration)

## Features

- Responsive navbar
- Hero section with typewriter effect
- Services, Projects, Testimonials, and Contact sections
- Simple animations and hover effects

## How to Run

1. Clone or download this repository
2. Open `index.html` in your browser

## Project Structure

```
Digital_marketing/
├── index.html
├── style.css
└── script.js
```
